# Amazon-Sales-Dashboard-using-Tableau-Desktop
Developed an interactive cybersecurity dashboard in Power BI to analyze global cyber attack trends, financial losses, and industry-wise impact from 2015–2024.
• Tools/ Techniques: Used data cleaning, transformation, DAX calculations, charts, maps, slicers, and data visualization techniques with Power BI and Excel.
• Outcome: Generated insights on major attack types, vulnerabilities, and affected industries to support better cybersecurity monitoring and decision-making.
